# Wandering Inn Extension

Shosetsu extension for reading [The Wandering Inn](https://wanderinginn.com) by pirateaba.


## Usage

Add `https://oneautumnmango.github.io/wanderinginn-extension/` as a repository in Shosetsu.
Download the `Wandering Inn` extension.

## Testing

```bash
./test-server.sh
```

Add `http://<your ip>:8000` to Shosetsu's repository list.
